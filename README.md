# Solar's Magnificent Pizza Collection Game!

<a href="https://solarcosmic.net/hc/pizzacollect/">Check out the demo here!</a>

 <img width="1672" height="1013" alt="Screenshot_20250909_222422" src="https://github.com/user-attachments/assets/d714d9ac-9e38-4448-9b0b-ef6b4aa1d45a" />

A simple game where you collect pizzas that fall from the sky! Made in Godot for Code Quest.

The base for the pizzas is +1. For every Mythic Pizza, you get +4, which includes the pizza base count, so +5.

I hope you enjoy this game! Feel free to leave feedback.
